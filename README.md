# 🔢 Number System Converter (DLD Project)

A specialized C++ utility developed to bridge the gap between **Digital Logic Design (DLD)** theory and practical software implementation. This repository showcases two distinct approaches to Command Line Interface (CLI) design.

---

### 📂 Project Evolution

#### 🚀 Pro Version (Animated & Stylized)
* **Interactive UX:** Features smooth screen transitions using `system("cls")`.
* **Visual Structure:** Uses custom borders and headers for a polished "Application" feel.
* **Focused Experience:** Clears previous data to keep the interface clutter-free.

#### 📝 Basic Version (Core Logic)
* **Minimalist Design:** Optimized for speed and performance.
* **Algorithm Focused:** Best for reviewing the raw mathematical logic behind base conversions.

---

### ✨ Core Functionalities

This tool provides seamless bi-directional conversion between the four major number systems:
* **Decimal** (Base 10)
* **Binary** (Base 2)
* **Octal** (Base 8)
* **Hexadecimal** (Base 16)

---

### 🛠️ Technical Highlights

* **Mathematical Precision:** Implements base-conversion algorithms from scratch without external math libraries.
* **Input Validation:** Intelligent error-handling to detect invalid inputs (e.g., entering 'A' in a Binary field).
* **Object-Oriented Approach:** Organized code structure for better maintainability.

---

### 🎓 Academic Context
Developed as a semester project for the **Digital Logic Design (DLD)** course at **Capital University of Science and Technology (CUST)**. The goal was to demonstrate mastery over number systems and base transformation logic.
